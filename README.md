# Campus Psych Lab

대학생 생활 데이터를 자기이해 인사이트로 연결하는 AI 기반 웹서비스

> 감정 기록, 심리테스트, 생활 칼럼을 하나의 흐름으로 연결해 대학생이 자신의 생활 패턴과 감정 변화를 가볍게 돌아볼 수 있도록 돕는 웹서비스입니다.

## 프로젝트 소개

Campus Psych Lab은 대학생의 일상에서 자주 등장하는 시험, 팀플, 연애, 집중력, 과제 미루기, 스트레스 같은 주제를 바탕으로 만든 자기이해형 웹서비스입니다.

현재 버전은 정적 HTML/CSS/JavaScript 기반으로 구현되어 있으며, 감정 기록소, 심리테스트, 생활 칼럼의 핵심 콘텐츠를 포함하고 있습니다. 향후에는 Next.js, Supabase/Firebase, AI 분석 API를 도입해 사용자의 기록과 테스트 결과를 기반으로 감정 변화 리포트, 키워드 추출, 생활 패턴 시각화, 맞춤형 콘텐츠 추천 기능을 제공하는 방향으로 발전시킬 계획입니다.

이 프로젝트는 의료적 진단이나 심리 상담을 대체하는 서비스가 아닙니다. 사용자가 자신의 일상 데이터를 더 잘 이해하고, 스스로를 관찰하는 데 도움을 주는 Human-centered AI 기반 자기성찰 도구를 지향합니다.

## 문제 정의

대학생은 시험, 과제, 인간관계, 진로 고민 등 다양한 스트레스 요인을 동시에 경험하지만, 자신의 감정과 생활 패턴을 꾸준히 기록하고 돌아볼 수 있는 계기를 갖기 어렵습니다.

일반적인 감정 기록 앱은 기록 자체에 집중하고, 심리테스트 서비스는 일회성 재미 요소에 머무는 경우가 많습니다. Campus Psych Lab은 이 둘을 생활 칼럼과 연결해 다음 질문에 답하고자 합니다.

- 나는 어떤 상황에서 감정 기복이 커지는가?
- 반복적으로 등장하는 생활 키워드는 무엇인가?
- 심리테스트 결과와 실제 감정 기록 사이에 어떤 패턴이 있는가?
- 나에게 도움이 될 만한 생활 콘텐츠는 무엇인가?

## 핵심 기능

### 감정 기록소

- 하루의 감정과 짧은 메모를 기록
- 감정 이모지와 태그를 함께 입력
- 최근 감정 기록 확인
- 주간 감정 변화와 감정 통계 UI 제공
- 익명 감정 공유 공간으로 공감 경험 제공

### 심리테스트

- 대학생 생활 주제 기반 테스트 제공
- 벼락치기, 연애, 팀플, 집중력, 과제 미루기 유형 테스트
- 선택지 기반 결과 유형 분류
- 결과별 해설 페이지와 연결

### 생활 칼럼

- 시험기간, 팀플, 집중, 연애, 미루기 등 대학생 생활 주제 칼럼
- 테스트 결과와 연결 가능한 콘텐츠 구조
- 향후 사용자 패턴 기반 추천 콘텐츠로 확장 가능

### 감정 통계 및 리포트

- 감정 점수 변화 시각화
- 시간대별, 요일별 감정 패턴 확인
- 자주 등장하는 태그와 생활 키워드 확인
- 향후 AI 기반 주간/월간 리포트로 확장

## 기존 프로젝트에서 발전시키려는 방향

현재 프로젝트는 정적 웹사이트 형태로 주요 페이지와 콘텐츠가 구성되어 있습니다. 포트폴리오용 서비스로 발전시키기 위해 다음 방향으로 리팩토링할 계획입니다.

- HTML 페이지 중심 구조를 Next.js 라우팅 기반 구조로 전환
- 반복되는 심리테스트 로직을 데이터 기반 테스트 엔진으로 통합
- 감정 기록 데이터를 localStorage 또는 정적 샘플이 아닌 DB에 저장
- 테스트 결과, 감정 기록, 칼럼 추천을 하나의 사용자 경험으로 연결
- 감정 변화와 생활 패턴을 차트와 리포트로 시각화
- AI 분석 결과를 사용자가 이해하기 쉬운 문장과 UI로 제공
- 단순한 기능 추가가 아니라, 기존 콘텐츠 방향성을 유지한 서비스 고도화

## AI 기능 계획

Campus Psych Lab의 AI 기능은 사용자를 평가하거나 진단하는 것이 아니라, 사용자가 자신의 기록을 더 쉽게 해석하도록 돕는 보조 도구로 설계합니다.

### 감정 분석

사용자가 작성한 감정 기록을 바탕으로 긍정, 중립, 부정뿐 아니라 불안, 피로, 성취감, 스트레스 등 일상적인 감정 신호를 분류합니다.

### 키워드 추출

감정 기록에서 반복적으로 등장하는 키워드를 추출합니다.

예시:

- 시험
- 팀플
- 친구
- 수면
- 연애
- 과제
- 알바

### 감정 변화 리포트

최근 7일 또는 30일 기록을 요약해 사용자가 자신의 흐름을 돌아볼 수 있도록 돕습니다.

예시:

> 이번 주에는 `시험`, `과제`, `피곤` 키워드가 자주 등장했어요. 주중 후반으로 갈수록 피로 관련 표현이 늘어났습니다.

### 맞춤형 칼럼 추천

감정 기록과 심리테스트 결과를 함께 참고해 관련 생활 칼럼을 추천합니다.

예시:

- 팀플 스트레스 기록이 많을 때: 팀워크 칼럼 추천
- 집중력 저하 키워드가 많을 때: 집중 루틴 칼럼 추천
- 시험 관련 불안 표현이 많을 때: 벼락치기/시험기간 루틴 칼럼 추천

### Human-centered AI UX 원칙

- 사용자를 단정하지 않고 가능성 중심으로 표현
- 의료적 진단처럼 보이는 문구 사용 지양
- 분석 결과에는 “참고용” 맥락을 함께 제공
- 사용자가 직접 기록을 수정하고 해석할 수 있는 여지를 남김
- 민감한 감정 데이터는 비공개를 기본값으로 설계

```txt
1. 사용자가 서비스에 접속한다.
2. 심리테스트를 통해 자신의 대학생활 유형을 확인한다.
3. 감정 기록소에서 하루의 감정과 상황을 기록한다.
4. 기록된 감정 데이터가 저장된다.
5. AI가 감정 흐름과 주요 키워드를 분석한다.
6. 사용자는 차트와 리포트로 자신의 패턴을 확인한다.
7. 분석 결과에 따라 관련 생활 칼럼을 추천받는다.
8. 사용자는 자신의 생활 루틴과 감정 변화를 지속적으로 돌아본다.
```

## 실행 방법

현재 프로젝트는 정적 HTML/CSS/JavaScript 기반입니다. 별도의 빌드 과정 없이 브라우저에서 실행할 수 있습니다.

### 로컬 실행

```bash
git clone https://github.com/your-username/campus-psych-lab.git
cd campus-psych-lab
```

이후 `index.html` 파일을 브라우저에서 열어 확인할 수 있습니다.

### 로컬 서버로 실행

정적 파일 경로 확인을 위해 간단한 로컬 서버 실행을 권장합니다.

```bash
npx serve .
```

또는 Python이 설치되어 있다면 다음 명령어를 사용할 수 있습니다.

```bash
python3 -m http.server 3000
```

브라우저에서 아래 주소로 접속합니다.

```txt
http://localhost:3000
```

## 프로젝트 방향성

Campus Psych Lab은 대학생의 감정과 생활 데이터를 수집하는 데서 끝나는 서비스가 아니라, 그 데이터를 사용자가 이해 가능한 인사이트로 되돌려주는 것을 목표로 합니다.

감정 기록은 개인의 하루를 남기는 작은 행동이고, 심리테스트는 자신을 가볍게 탐색하는 입구이며, 생활 칼럼은 다음 행동으로 이어지는 콘텐츠입니다. 이 세 가지를 AI 기반 분석과 연결해 “나를 판단하는 서비스”가 아니라 “나를 더 잘 관찰하도록 돕는 서비스”로 발전시키는 것이 이 프로젝트의 핵심 방향입니다.
